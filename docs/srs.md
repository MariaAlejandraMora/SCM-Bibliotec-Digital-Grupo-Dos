## HU-02 – 💻 Solicitar Préstamo Digital

Como *usuario registrado* quiero solicitar el préstamo de un libro digitalpara poder leerlo en línea

### Criterios de Aceptación
- Solo usuarios autenticados pueden solicitar préstamo.
- El sistema debe verificar disponibilidad del libro.
-	Debe generarse confirmación de préstamo.

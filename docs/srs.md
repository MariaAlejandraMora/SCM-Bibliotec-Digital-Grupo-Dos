# HU-01 – Registro de Usuario
## Historia de Usuario
**Como** visitante  
**Quiero** registrarme con correo electrónico y contraseña  
**Para** poder acceder a la plataforma  
## Criterios de Aceptación
- El correo no debe estar previamente registrado.
- El sistema debe validar el formato del correo electrónico.
- Debe mostrarse un mensaje de registro exitoso.
